# Caputo-Riemann-Liouville-vpa
MATLAB codes accompanying:

Francisco de la Hoz and Peru Muniain, Numerical approximation of Caputo-type advection–diffusion equations via shifted Chebyshev polynomials (2025).

If you use this software in your research, please cite the above reference.

The programs pde2D.m and pdeND.m require the MATLAB functions herdif, herroots, and poldif, which are part of the MATLAB differentiation suite developed by J. A. C. Weideman and S. C. Reddy. These functions are available at:

https://appliedmaths.sun.ac.za/~weideman/research/differ.html
